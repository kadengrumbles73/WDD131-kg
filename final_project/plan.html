Kaden Grumbles

WDD 131-01

Overview

Purpose

The purpose of this site is to provide NFL fans and data enthusiasts with a clean, functional dashboard to view player statistics and team performance metrics. It serves as a frontend interface for a custom data pipeline that aggregates football data into a readable format.

Audience

NFL fans and data enthusiasts

Dynamic elements

I will use JavaScript to allow users to filter player lists by position without reloading the page and to automatically display the current year and the last modified date of the document in the footer to ensure data transparency.
